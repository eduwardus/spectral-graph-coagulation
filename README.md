# Spectral Graph Coagulation

Code, processed data, figures, and manuscript sources for the project:

**Spectral Graph Coagulation and the Emergence of Hybrid Discrete Geometry**

## Repository structure

- `src/` — Python scripts for simulation, analysis, and figure generation
- `data/` — processed data files used in the analyses
- `images/` — final figures used in the manuscript
- `paper/` — LaTeX source of the manuscript

## Main results

The repository supports analyses related to:

- effective coagulation kernel
- topological scaling
- spectral dimension
- metric growth
- eigenvector structure
- discrete curvature
- emergent hybrid geometry
