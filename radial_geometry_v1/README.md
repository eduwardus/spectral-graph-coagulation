# Radial Geometry in Spectral Graph Coagulation

This repository contains the code and processed data used in the paper:

**"Coarse-grained radial structure and scaling in spectral graph coagulation"**

## Structure

- `scripts/` → Python scripts (Blocks 1–9)
- `data/` → Input and processed data
- `figures/` → Figures in the paper
- `results/` → Intermediate outputs

## Reproducibility

The figures in the paper can be reproduced by running:

1. radial decomposition (Block 1–2)
2. curvature and transport (Block 3–5)
3. collapse analysis (Block 6–9)

## Notes

- Large raw simulation files are not included.
- Processed datasets are provided instead.
